## Overview

GlowGuide is a **web-based cosmetics recommendation system** built with **Python and Flask**, allowing users to receive **personalized cosmetic suggestions** based on their **skin type, budget, and product preferences**. The platform uses **machine learning models** to provide accurate recommendations and features a **responsive, modern UI** for seamless user interaction.

---

## Key Features

* **Skin Type Selection:** Users can select their skin type or take a quick test to determine it.
* **Budget-Based Recommendations:** Users can choose a budget range to filter recommended products.
* **Subcategory Selection:** Allows users to refine product recommendations by specific cosmetic subcategories.
* **Hybrid Recommendation System:** Uses **pre-trained ML models** (Scikit-learn) for personalized product suggestions.
* **Responsive Design:** Works smoothly on desktop, tablet, and mobile devices.

---

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **Machine Learning & Data Processing:** 
  - Scikit-learn
  - Pandas
  - NumPy
  - NLTK
  - SpaCy
  - Joblib




## Machine Learning Component

* **Model Type:** Hybrid recommendation system (Rule-based + ML)
* **Libraries:** Scikit-learn, Pandas, NumPy
* **Function:** Predicts suitable cosmetic products based on **skin type, subcategory, and budget**.
* **Data Handling:** Preprocessed datasets stored as Python objects; no external database required.
